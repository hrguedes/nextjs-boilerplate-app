## Running

### Pre requisites

- NodeJs
- NextJs

Open terminal in folder project and running for install packages

```bash
    npm i
```

configure file `.env` in root folder

```env
    API_BASE_URL=
    JWT_SECRET=
```

running project
```bash
    npm run dev
```
